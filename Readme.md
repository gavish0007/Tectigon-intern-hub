Tectigon Intern Hub
A modern and responsive Intern Management Dashboard built using HTML, CSS, and JavaScript.
Overview
Tectigon Intern Hub is a centralized platform designed to help interns manage their daily activities, tasks, attendance, standups, resources, team information, and profile details through an intuitive dashboard interface.
